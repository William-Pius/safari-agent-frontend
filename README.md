# SAFARI Agent Frontend

Frontend for SAFARI Agent, a platform for booking bus tickets and transporting items.

## Features

- Booking tickets for traveling
- Booking tickets for transporting items
- Scheduling routes
- Making payments
- Customer support

## Technologies Used

- React Native
- Redux
- Axios for API calls
- React Navigation
